# **App Name**: CineScript AI

## Core Features:

- Story Idea Input: Users provide core elements such as character names/count and a basic movie premise via a structured input form.
- AI Plot Generation Engine: Utilizes the Gemini 3 Flash model as a cinematic tool to transform high-level input into detailed, multi-scene storyboards, adhering to professional cinematography standards.
- Interactive Storyboard Display: Presents the generated scenes in a clear, scrollable interface, explicitly demarcating each panel and the associated image generation prompt.
- In-app Scene Editor: Provides an editable text area for each scene's details, enabling users to fine-tune descriptions and image prompts before finalization.
- Scene Management & Export: Allows users to confirm edits and export individual scenes as 'scene-i.md' markdown files into a designated 'scenes' folder, facilitating project saving.
- Gemini API Key Configuration: Enables secure configuration of the Gemini API key through a local '.env' file, ensuring seamless access to the AI model for content generation.

## Style Guidelines:

- A sophisticated dark theme inspired by cinematic post-production suites. Primary interactions and highlights utilize a modern blue (#407DFA), evoking digital creativity. The deep, muted background (#1C1E23) provides excellent contrast and reduces eye strain.
- Accent elements and notifications are highlighted with a vibrant cyan (#66DAF0), reminiscent of diagnostic displays or future-tech interfaces, providing clear visual cues.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and bold appearance, fitting the generative AI theme. Body text font: 'Inter' (sans-serif) for high legibility and a neutral, professional feel suitable for detailed plot descriptions and user input.
- Minimalist, clean vector icons with a focus on geometric shapes, designed to integrate seamlessly with the dark, professional interface. Icons should represent common film and editing metaphors.
- A multi-panel, responsive layout optimized for creative workflows. Features a prominent input section and a scrollable output display, resembling professional film editing or storyboarding software interfaces. Generous use of padding and whitespace for visual clarity.
- Subtle, smooth transitions and fades for loading states and scene changes. Micro-animations on interactive elements (buttons, inputs) provide gentle feedback without distracting from the creative process.
